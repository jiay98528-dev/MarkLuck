## 2026年5月15日 晴

今天起得很早，天还没亮透。窗外有鸟叫声，不知道是什么鸟，但叫得挺有节奏。泡了杯速溶咖啡，坐在书桌前翻了几页昨天没看完的书。看到一半想起昨天还有一个 bug 没修完，心里就有点不踏实。

上午花了三个小时重构那个 API 模块。本来以为很简单，结果越改越多。先是发现参数校验的逻辑写得太分散，然后发现错误处理在不同层里各做各的。最后索性把整个 validation 层抽出来单独写了一个中间件。改完之后测试全绿，心里松了一口气。

中午下楼买了份盖饭。老板娘问我怎么最近都没来，我说在家赶项目。她多给我加了个卤蛋。下午回来继续写文档。写 API 文档的时候发现——其实写代码和写文章很像，都需要在"说清楚"和"说太多"之间找个平衡。

晚上跑步五公里。跑完回来冲了澡，坐在阳台上吹风。想起以前在上一家公司的时候，加班到凌晨是常态。现在虽然也忙，但至少是自己选择的方向。这种掌控感挺珍贵的。

## 2026年5月18日 阴

天气转凉了。换上了长袖。

今天收到一封邮件，之前投的简历有回音了。是一家做开源工具的小公司，技术栈正好是我熟悉的。约了下周三面试。说不上紧张，但也不轻松——面试这种事情，准备再多都感觉不够。

下午在整理笔记的时候突然想到一个点：我过去半年的技术积累，大部分都散落在各种草稿和临时文档里。没有一个系统性的知识库。于是决定用 Markdown 把重要的东西重新梳理一遍。用 `[[` 链接把相关的概念串起来，用 `#tag` 做分类。整理了三篇笔记，发现思路反而比以前更清楚了。果然写作能帮助思考。

晚上给家里打了电话。妈妈说她最近学会了用手机买菜，挺开心的。爸爸在旁边插话说让我别太累了。挂了电话之后发了会儿呆。其实他们一直不太懂我具体在做什么，但支持是真心的。

## 2026年5月22日 雨

下雨天适合窝在家里写代码。

今天完成了一个功能模块的全部单元测试。从早上九点写到下午四点，中间除了吃饭就没离开过椅子。站起来的时候腰有点酸。写测试这件事吧——写的时候觉得枯燥，跑通的时候又觉得很有成就感。特别是发现了一个之前完全没注意到的边界情况。

测试覆盖率到了百分之九十之后，开始优化性能。用 Chrome DevTools 的 Performance 面板跑了几次，发现渲染管线里有一个不必要的重排。改了之后页面 FPS 从 30 提升到了 55。几行代码的改动，体验改善很明显。

晚上看了半部电影，没看完就困了。明天继续。

## 2026年5月28日 多云

周末。睡到自然醒。

上午去书店逛了一圈。买了三本书：一本关于软件架构的、一本散文集、还有一本讲中国书法史的。书店里人不多，背景音乐是爵士钢琴。这种环境特别适合慢慢翻书——不是那种功利性的"找答案"，而是纯粹的"随便看看"。

下午约了朋友喝咖啡。他刚从一个创业公司离职，聊了很多关于技术选型的话题。他说之前那个项目最大的问题不是技术，而是方向一直在变。每两个月 pivot 一次，代码写得再好也没意义。我觉得他说得对。

晚上回到家，打开了前几天写了一半的那篇技术文章。重新读了一遍，删了大概三分之一的废话。写文章最大的敌人就是自己舍不得删。又加了一个实际案例，文章终于看起来像样了。明天再校对一遍应该就能发了。

## 2026年6月3日 晴

今天正式开源了我的项目。

把 README 写完了——中英文各一版。写英文版的时候特别小心，生怕语法和用词不地道。用 Grammarly 检查了三遍。配置了 CI/CD，写好了贡献指南，加上了 MIT License。最后一刻按下"Publish"按钮的时候，手竟然有点抖。

一个小时后收到了第一个 Issue。不是 bug，是有人问怎么安装。赶紧补了一节安装指南。然后又收到一个 Star——虽然不知道是谁，但突然觉得这件事是有意义的。

晚上复盘了一下今天的收获。最大的感悟是：开源不只是把代码放出去。它是写文档、做配置、回复问题、解释设计决策的全过程。代码只占其中的一部分。今天累了，但很满足。
