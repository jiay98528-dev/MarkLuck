# JotLuck 商业计划与发行作战页

更新时间：`2026-07-10`

## 文件说明

- `index.html`：单文件静态长页，面向内部执行，可直接双击离线打开
- `sources.md`：本页使用的仓库内事实与外部公开来源清单

## 这份页面回答什么

这不是融资 BP，也不是夸张 TAM 的增长故事。

它只回答 4 个更现实的问题：

1. 以 JotLuck 当前真实状态，首年应该怎么赚钱
2. 中国大陆两人公司，用什么路径收款最稳
3. 接下来 12 个月，哪些节点必须按顺序完成
4. 宣传与获客应该优先投哪些渠道，哪些暂时不要碰

## 适用前提

- 项目品牌：`JotLuck`
- 当前对外状态：`Windows x64 / v0.15.0-rc.1 / 公开 Beta 或 RC 导向`
- 产品边界：`MIT 开源`、`本地优先`、`纯文本笔记`、`无云同步`、`无多人协作`、`无应用内支付`
- 团队假设：`中国大陆两人公司（1 开发 + 1 商务）`

## 使用方式

1. 直接打开 [index.html](./index.html)
2. 断网也可完整查看
3. 需要核对证据时，查看 [sources.md](./sources.md)

## 研究口径

- 所有价格、政策与平台规则统一按 `2026-07-10` 口径记录
- 内部事实优先引用仓库内文档
- 外部事实优先使用官方首页、定价页、帮助中心、开发者文档、税务公告
- 页面会明确区分：
  - `事实`
  - `判断`
  - `建议动作`

## 不覆盖内容

- 融资故事
- 宏观 TAM 估算
- 未落地移动端、协作云、真实主题市场
- 需要律师正式出具意见的法律结论
