## 证据分级

| 来源类型                            | 级别         | 用法                                 |
| ----------------------------------- | ------------ | ------------------------------------ |
| 仓库内产品/发行文档                 | `core`       | 确认当前产品事实、发行状态、功能边界 |
| 官网首页 / 定价页 / 下载页          | `core`       | 确认定位、价格、分发方式             |
| 隐私 / 条款 / 帮助中心              | `core`       | 确认支付、退款、数据边界、规则       |
| 官方开发者文档 / 税务公告           | `core`       | 确认支付接入、开票、平台规则         |
| 官方博客 / 创作者手册 / GitHub Docs | `supplement` | 补充最佳实践、创作规则和分发语境     |
| 社区讨论 / 媒体转述                 | `avoid`      | 不作为核心事实依据                   |
