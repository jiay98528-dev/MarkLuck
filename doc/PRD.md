# MarkLuck PRD

版本：2026-06-26

## 产品定位

MarkLuck 是一款本地优先、离线可用的 Markdown 笔记工具。数据以普通文件存在，文件夹即笔记本，应用负责提供稳定的编辑、检索、预览、导出与导航体验。

## 当前主题约束

- 默认官方主题仍为 `paper` / “羽翼布局”，但主题系统必须支持本地市场、安装、预览、启用、卸载和回退。
- 主题以 `ThemeManifest v2 + ShellRecipe + UxComponentRecipe` 为主协议，可控制 Shell 布局、区域、动作路由、视觉 token、资产和动效。
- 声明式主题不得执行任意代码；`official-code` 和经用户授权的 `trusted-code` 主题可通过受控 `ThemeHostApi` 替换 Shell 级 UX 组件。
- 网络、文件系统读写、搜索索引、Markdown 安全清洗、导出服务和系统 API 不向主题默认开放。

## 核心用户能力

- 打开本地文件夹，浏览 Markdown / 文本笔记。
- 在编辑器中进行实时编辑与自动保存。
- 使用即时预览、Wiki-link、反向链接、标签与大纲导航。
- 通过搜索、模板、导出和自动补全提升工作效率。
- 在欢迎引导、设置、文件抽屉、命令面板等外围流程中保持一致的单布局体验。

## 非目标

- 额外外观运行时
- 额外配色切换
- 第三方外观扩展
- 在线外观市场

## 验收基线

- 启动后默认应用 `paper` / 羽翼布局。
- 刷新、切换笔记、打开设置后，布局保持一致。
- 设置页不出现额外外观切换入口。
- 欢迎页、首页、文档、测试中不再引用已移除的外观扩展能力。
