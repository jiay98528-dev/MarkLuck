| 来源类型                 | 级别         | 用法                                 |
| ------------------------ | ------------ | ------------------------------------ |
| 官网首页                 | `core`       | 确认定位、目标用户、主功能叙事       |

| 隐私/条款页              | `core`       | 确认数据边界、联网说明、退款/授权    |
| 官方博客                 | `supplement` | 补充版本、路线图与公告语境           |
| GitHub README / Releases | `supplement` | 补充开源状态、更新节奏、发行说明     |
| App Store / 应用商店页   | `supplement` | 补充平台信息和公开商店分发情况       |
| 第三方媒体 / Reddit      | `avoid`      | 不用于核心事实，仅用于噪声或舆情观察 |
